The z-index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order. Note: z-index only works on positioned elements (position:absolute, position:relative, or position:fixed). Default value: auto Inherited: no Animatable: yes. Read about animatable Version: CSS2 JavaScript syntax: object.style.zIndex="-1" Browser Support The numbers in the table specify the first browser version that fully supports the property.
